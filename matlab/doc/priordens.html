SYNOPSIS

function lnprior = priordens(para, pshape, p1, p2, p3, p4)

DESCRIPTION 

 This procedure computes a prior density for
 the structural parameters of the DSGE models
 pshape: 0 is point mass, both para and p2 are ignored
         1 is BETA(mean,stdd)
         2 is GAMMA(mean,stdd)
         3 is NORMAL(mean,stdd)
         4 is INVGAMMA(s^2,nu)
         5 is UNIFORM [p3,p4]

CROSS-REFERENCE INFORMATION

This function calls:
  • lpdfgam log GAMMA PDF
  • lpdfgbeta log (generalized) BETA PDF
  • lpdfig1 log INVERSE GAMMA (type 1)
  • lpdfig2 log INVERSE GAMMA (type 2)
  • lpdfnorm LPDFNORM The log of the normal density function
